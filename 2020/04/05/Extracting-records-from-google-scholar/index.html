Extracting records from Google Scholar

Google Scholar Web Crawler Beautifulsoup
Python

阅读次数:  

Usually, we like to search academic paper using Google Scholar, but sometimes, it’s an annoying thing that if we want to download all of the records mannually. And we may asking that dose anything can help improving the efficiency? The answer is yes! The following code helps you extracting usefull information about what you’ve searched in google scholar, including title, author, journal, year and cited amount.

1.Loading modules

import requests
import urllib.parse

import random

import time

    timeout = random.choice(range(180, 210))  # prevent blocking IP
    Res = requests.get(url, headers=headers, timeout=timeout).content
    soup = BeautifulSoup(Res, "lxml")

    Papers_infor = soup.find_all('div', {'class':'gs_ri'})


    for paper in Papers_infor:

        title = paper.find('h3', {'class','gs_rt'})
        #if title.a is not None:

        #else:
        #    TITLES.append("without title 404!!")

        cite = paper.find('div',{'class':'gs_fl'})
        cite1 = cite.get_text()
        cite2 = re.findall(r"\d+\.?\d*",cite1) # Extracting cited amount

        author = paper.find('div',{'class':'gs_a'})
        author1 = author.get_text()

        AUTHORS.append(author1)
        CITES.append(cite2)
        TITLES.append(title.a.get_text())


    time.sleep(3) # interval time was 3 seconds
    print(f"have finished {i+1} pages")



##### output the csv file #####
PAPERS_INFOR = pd.DataFrame({

    'title': TITLES,
    'author': AUTHORS,
    'cited_num': CITES

})

PAPERS_INFOR.to_csv("Records.csv", index=False)

3.Conclusion

Please seting loop interval time more than 3 seconds, and it’s a safe way to continuely download your recosrd!
